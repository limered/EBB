Ornament im Raum

Das Projekt "Ornament im Raum" entstand als Zusammenarbeit zwischen dem Kunstbereich der Universität Koblenz und dem b-05 Kunst- und Kulturzentrum in Montabaur unter der Leitung von Dr. Markus Lohoff.

Das Ziel war es Demoprogramme zu erstellen, die Formsprache und Eigenschaften von Ornamenten verwenden und diese in ein digitales Zeitalter transferieren. Als Zusatzanforderung sollte Randomisierung verwendet werden um Abwechslung zu schaffen. Grundsätzlich wurde Java als Programmiersprache zusammen mit dem Processing Framework verwendet.

Mein Beitrag bestand aus zwei Programmen, die auf dieser Seite vogestellt werden. Meine Demos sind hier in Orginalform als Video zu sehen. Die hier verlinkten Echtzeitdemos sind unter verwendung von WebGL und HTML5 entstanden. Fürs Rendering wurde THREE.js verwendet. Daneben wurden die Demos auf Abspielen in Chrome optimiert, sollten aber auch ohne Probleme im Firefox funktioniern.

Zusatzarbeiten

Kamera Klasse

Um Benutzereigaben abgreifen zu können musste eine Webcam angesprochen werden. Die entsprechende Klasse wurde von mir implementiert. Dort werden die aufgenommenen Bilder auf Bewegungen gescannt. Dafür werden immer die letzten Zwei Bilder miteinender verglichen und die sich veränderten Pixel extrahiert. Um debei den Rechenaufwand gering zu halten, werden die Pixel in Metapixel zusammengefasst. Die Größe der Metapixel wird vor Programmstart definiert und spiegelt die Genauigkeit, mit der Bewegungen erkannt werden wieder. Als letzen verarbeitungsschritt werden die Metapixel zu Inseln (Blobs) zusammengefasst, deren Mittelpunkt an das Program zurückgegeben wird.

Sound

Die musikalische Untermalung beider Demos wurde von mir mit einem einfachen Musikprogramm erstellt. Bei den Linien ist die Musik ein elemantarer Teil der Demo. Hohe Ausschläge bestimmter Frequenzten wurden dazu verwendet, die Partikel aufläuchten zu lassen. Im Browser wird dazu die HTML5 Web Audio Api verwendet.
